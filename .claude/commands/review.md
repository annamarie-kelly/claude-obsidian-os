# Weekly Review

The weekly metacognition pass. Assumes triage and loops have already been run this week — this is about synthesis, not processing.

$ARGUMENTS

## Steps

1. **Scan recent activity** — check all notes modified in the past 7 days across the vault. Summarize what was worked on by domain (Building, Thinking, Working, Relating).

2. **Surface emerging patterns** — from recent notes, identify insights worth distilling:
   - New patterns to extract (suggest claim-style titles)
   - Seed patterns ready to promote to `growing`
   - Growing patterns ready to promote to `evergreen`
   - Suggest updates to relevant `_patterns.md` files

3. **Project health** — for each active project in `01-Building/`, check if recent notes mention it. Flag projects with no activity this week.

4. **Orphan check** — find notes with zero inbound links that might need connecting. Suggest specific `[[wikilinks]]` to add.

5. **Inbox check** — if there are unprocessed items in `00-Inbox/`, flag them (but don't process — that's `/triage`).

6. **Create review note** — save a summary to `07-Archive/` using the Weekly Review template with:
   - What domains got attention this week
   - Patterns extracted or promoted
   - Connections made
   - What felt neglected

Present findings as a summary and ask before making changes.
