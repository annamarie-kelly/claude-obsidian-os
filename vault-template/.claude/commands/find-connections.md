# Find Connections

Scan the vault for notes that should be linked but aren't.

$ARGUMENTS

1. **Read the target note** (if specified via argument) or scan all notes modified in the past 7 days.
2. **Search for related content** across `01-Building/`, `02-Thinking/`, `04-Living/`, `05-Relating/`.
3. **Check `_patterns.md` files** — does the note fit into an existing patterns section?
4. **Suggest new `[[wikilinks]]`** to add — show exactly where in each file.
5. **Flag orphans** — notes with zero inbound links that might need connecting.
6. **Cross-domain connections** — look for links between domains. A ceramics insight might connect to a person who taught it. A reading note might inform an active project.

Present suggestions and ask before making changes.
